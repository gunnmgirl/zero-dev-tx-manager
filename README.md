## Sending gasless transactions!

Showcasing ZeroDev's Paymaster service by utilizing gasless transactions.\
Check more on Paymaster service [here](https://docs.zerodev.app/smart-wallet/sponsoring-gas).

## Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
